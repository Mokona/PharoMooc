\chapter{SUnit Testing}

\mainauthor{St\'ephane Ducasse}

For each class described in the Chapter on collection "Set, Dictionary and Bag", define unit tests for each of the described behavior. 
 
\section{Set}

\exercise create a Unit test for each of the following methods

\begin{itemize}
\item \ct{aSet add: anObject} and \ct{aSet addAll: aCollection},
\item \ct{aSet includes: anObject}
\item \ct{aSet remove: un Objet} and \ct{aSet remove: anObjet ifAbsent: aBlocException},
\item  \ct{aSet size} and \ct{aSet capacity}. 
\end{itemize}


\section{Dictionary}

\item \ct{aDictionary includesKey: aKey},
\item \ct{aDictionary includes: anObject},
